Site vitrine généré pour "Planète Jupiter", vidéaste.

**Stack:** Next.js 14 (App Router) · Tailwind CSS · GSAP + ScrollTrigger · Three.js (planète uniquement)

**Direction artistique:** Noir & blanc strict, typographie Cormorant Garamond (display) + Inter (body), grain cinéma CSS, glow Fresnel Three.js.

**Structure pages:**
- `/` — Landing immersive (Hero + Manifeste + Projets + Showreel + About + Contact CTA)
- `/portfolio` — Grille éditoriale avec filtres par catégorie
- `/contact` — Formulaire complet (à brancher sur un service email)
- `/mariage`, `/entreprise`, `/clip` — Pages stubs prêtes à remplir

**Texture Jupiter:** `/public/textures/jupiter.jpg` (copiée depuis `assets/2k_jupiter.jpg`)

**SEO:** metadata Next.js, Schema.org ProfessionalService, Open Graph, HTML sémantique.

**How to apply:** Pour toute modification, respecter la palette noir/blanc stricte et les animations lentes (ease power3, duration 1.2+). Ne jamais introduire de couleurs vives.

**Why:** Identité visuelle forte, immersive et cinématique — tout ajout doit préserver cet équilibre.
